About ODLeave Connect

ODLeave Connect is a web-based leave and On-Duty (OD) request management system developed to simplify the process of applying, tracking, and approving student requests.

Students can securely log in, submit OD or Leave requests, and monitor the approval status in real time. Faculty members can review requests and take appropriate actions by accepting or rejecting them through a dedicated dashboard.

Key Features

  • Student Login System
  • OD Request Management
  • Leave Request Management
  • Faculty Approval Dashboard
  • Request Status Tracking
  • PDF Approval Letter Generation
  • SQLite Database Integration

Project Objective

The objective of this project is to reduce manual paperwork, improve communication between students and faculty, and provide a simple digital solution for managing OD and Leave requests.
